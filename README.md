# 🔐 ReKampus Escrow DApp

A decentralized escrow smart contract built on **Stellar Soroban**, designed to enable secure and trustless transactions—especially within student communities.

---

## 📌 Overview

**ReKampus Escrow DApp** is a blockchain-based escrow system that removes the need for intermediaries in peer-to-peer transactions. By leveraging **Stellar's Soroban smart contract platform**, this project ensures that funds are only released when both parties fulfill their obligations.

---

## 🎯 Objectives

- Eliminate reliance on centralized intermediaries
- Ensure fairness in buyer–seller transactions
- Reduce fraud in student marketplaces
- Enable trustless financial interactions
- Provide transparent and verifiable transaction records

---

## ⚙️ Core Features

### 🧾 Create Escrow
Initialize a transaction by specifying Buyer, Seller, Token, and Amount.

### 💰 Deposit Funds
Buyer deposits funds into the escrow contract.

### ✅ Confirm Transaction
Buyer confirms successful delivery → funds released to seller.

### ❌ Cancel Escrow
Buyer can cancel the transaction → funds refunded.
